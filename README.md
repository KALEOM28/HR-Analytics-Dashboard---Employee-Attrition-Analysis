# HR Analytics Dashboard - Attrition Analysis

## 📌 Project Overview
This interactive Power BI dashboard provides a data-driven analysis of employee turnover within the organization. The primary objective is to help HR professionals and management discover key patterns, demographic factors, and satisfaction levels that lead to employee attrition, enabling them to implement effective retention strategies.

---

## 📊 Key Performance Indicators (KPIs)
The dashboard tracks the following core metrics to give a high-level summary of the workforce:
* **Total Employees (Total Emp):** Total headcount recorded in the dataset.
* **Active Employees (Active Emp):** Total number of current employees.
* **Attrition Count:** The total number of employees who have left the company.
* **Attrition Rate:** The percentage of overall employee turnover.
* **Avg Age:** The average age of the workforce.

---

## 🎛️ Interactive Filters & Slicers
The dashboard includes dynamic filters to slice and dice the data by:
* **Department-wise Analysis:** Filter insights across **Human Resources (HR)**, **Sales**, and **Research & Development (R&D)**.
* **Gender Overview:** Dynamic employee headcount split by **Male** and **Female**.
* **Marital Status:** Segregate retention trends based on **Married**, **Single**, and **Divorced** employees.

---

## 📈 Visualizations Included

### 1. Attrition by Tenure (Line Chart)
* Visualizes employee attrition trends against the number of years spent at the company.
* **Insight:** Helps identify the critical "risk years" where employee turnover is highest.

### 2. Age-wise Attrition (Column Chart)
* Breaks down the turnover numbers into different age brackets.
* **Insight:** Pinpoints which age generations require better engagement and retention efforts.

### 3. Attrition by Education (Donut Chart)
* Displays how employee attrition is distributed across various educational fields (e.g., Life Sciences, Medical, Marketing, Technical Degree).

### 4. Attrition by Job Role & Job Satisfaction (Stacked Bar Chart)
* Maps attrition counts across different job designations while overlaying **Job Satisfaction Rates (scaled from 1 to 4)**.
* **Insight:** Correlates workplace satisfaction directly with turnover across specific operational roles.

---

## 🛠️ Tech Stack Used
* **Data Visualization:** Power BI Desktop
* **Data Transformation & ETL:** Power Query
* **Data Modeling:** DAX (Data Analysis Expressions) for custom KPIs and dynamic metrics.

---

## 🚀 How to Explore the Dashboard
1. Clone or download this repository.
2. Open the provided `.pbix` file using **Power BI Desktop**.
3. Use the interactive slicers at the top and side panels to filter data dynamically.
